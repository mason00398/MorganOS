🏗️ Architecture

runst_x/
├── bootloader/          ← UEFI boot loader with SHA256 verification
│   ├── Cargo.toml
│   └── src/main.rs
├── kernel/              ← Main kernel source
│   ├── Cargo.toml       ← Dependencies: simple-ahci, fat32, fringe, etc.
│   ├── build.rs
│   └── src/
│       ├── main.rs      ← Kernel entry point, global driver instances
│       ├── panic.rs     ← Custom panic handler
│       ├── console.rs   ← Shell with history, tab completion, arrow keys
│       ├── pci.rs       ← PCI enumeration with bridge support
│       ├── drivers/     ← Hardware drivers
│       │   ├── ahci.rs      ← Simple-AHCI storage
│       │   ├── process.rs   ← Fringe-based preemptive scheduling
│       │   ├── fat32.rs     ← FAT32 filesystem driver
│       │   ├── rtl8139.rs   ← RTL8139 Ethernet with ARP cache
│       │   ├── tcpip.rs     ← TCP/IP stack with ICMP
│       │   ├── keyboard.rs  ← PS/2 keyboard (pc-keyboard crate)
│       │   ├── memory.rs    ← 16MB heap allocator
│       │   ├── net.rs       ← ARP resolution
│       │   ├── vga.rs       ← VGA text buffer
│       │   └── rtc.rs       ← Real-time clock
│       └── commands/    ← Shell commands (20+ built-in)
└── AUDIT_REPORT_v0.4.md ← Full audit report
